This directory contains code for ./bin/bbseg.

bbseg_joint.hpp
	The code does the real things.

bbsegj_main.cpp
	The code call bbseg_joint.hpp for each frame and each camera.
	Main function for ./bin/bbsegj is here.

bbsegj_test.cpp
	The code for ./bin/bbsegj_test.
	Because ./bin/bbsegj is designed for mpi, this is a non-mpi one frame version for debug purpose.
	This file also contains some sandbox code.
